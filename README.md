## Car Rental Management System (CRMS)

A Java based system that simulates the complete car rental lifecycle, including booking, processing, and finalization
using object oriented design principles.

---

### ⚙️ Features

- **Booking System:**
    - Create rental bookings linking clients, cars, and agents
- **Processing Module:**
    - Apply insurance options and discounts
    - Generate processed rental records
- **Finalization:**
    - Complete rental transactions and produce final records
- **Object Oriented Design:**
    - Clear separation of entities (Client, Car, Agent, Booking, etc.)
- **System Simulation:**
    - Mimics real world rental workflow step by step

---

### Project Structure

- Client.java
- Car.java
- Agent.java
- BookingRecord.java
- ProcessedRecord.java
- RentalSystem.java
- CarRentalImplementer.java

### Technologies Used
